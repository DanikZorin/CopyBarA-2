## Сборка и запуск

```bash
make          # сборка проекта
make run      # запуск программы
make clean    # очистка объектных и исполняемых файлов
```

## GitHub Actions

Проект использует GitHub Actions для:
- Автоматической сборки при push и pull requests
- Запуска unit-тестов (Google Test)
