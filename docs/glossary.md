# Глоссарий

| Термин | Описание |
|--------|----------|
| **Автомастерская** | Предприятие, оказывающее услуги по ремонту и обслуживанию автомобилей |
| **Мастер** | Сотрудник автомастерской, выполняющий ремонтные работы |
| **Заказ** | Запись о выполнении ремонтной работы для конкретного автомобиля |
| **Вид ремонта** | Тип выполняемых работ (замена масла, регулировка клапанов и т.д.) |
| **Госномер** | Уникальный идентификационный номер автомобиля |
| **Техпаспорт** | Документ, удостоверяющий регистрацию автомобиля |
| **Владелец** | Физическое лицо, которому принадлежит автомобиль |
| **Базовая стоимость** | Стандартная цена конкретного вида ремонта (хранится в справочнике) |
| **Итоговая стоимость** | Базовая стоимость с учётом скидки (возможной) |
| **CRUD** | Create, Read, Update, Delete — основные операции с данными |
| **SQLite** | Реляционная база данных, используется для хранения данных |
