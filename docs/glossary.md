# Глоссарий

| Термин | Описание |
|--------|----------|
| **Автомастерская** | Предприятие, оказывающее услуги по ремонту и обслуживанию автомобилей |
| **Мастер** | Сотрудник автомастерской, выполняющий ремонтные работы |
| **Заказ (Orders)** | Запись о выполнении ремонтной работы для конкретного автомобиля |
| **Вид ремонта (Repairs)** | Тип выполняемых работ (замена масла, регулировка клапанов и т.д.) |
| **Госномер** | Уникальный идентификационный номер автомобиля |
| **Техпаспорт** | Документ, удостоверяющий регистрацию автомобиля |
| **Владелец (Users)** | Пользователь системы, имеющий логин и пароль |
| **Базовая стоимость** | Стандартная цена конкретного вида ремонта (хранится в справочнике) |
| **Итоговая стоимость** | Базовая стоимость с учётом скидки (возможной) |
| **CRUD** | Create, Read, Update, Delete — основные операции с данными |
| **SQLite** | Реляционная база данных, используется для хранения данных |
| **Callback** | Функция обратного вызова для обработки результатов SQL-запроса |
| **Prepared statement** | Подготовленное выражение SQL с плейсхолдерами (`?`) |
| **Authenticator** | Класс, отвечающий за аутентификацию пользователя |
| **Server** | Класс, обрабатывающий команды и запросы к БД |
| **Date** | Класс для работы с датами в формате ГГГГ-ММ-ДД |
| **MAX_FAIL** | Константа (4) - максимальное количество попыток входа |
