# Проект: Система управления автомастерскими (Workshops)

## О проекте
Консольное приложение для автоматизации работы автомастерских с поддержкой SQL-запросов.

## Основные команды

| Команда | Описание |
|---------|----------|
| `-help` | Справка по командам |
| `-tables` | Список таблиц |
| `-show [table]` | Показать таблицу |
| `-sql [query]` | Выполнить SQL |
| `-1` ... `-5` | Задания 1-5 |
| `-p5`, `-p6` | Дополнительные задачи |
